KPRenameTitle
You can rename tab titles based on the URL. This is useful to support the auto-type functionality of KeePass.
Note that it might help to use the carrot (^) at the beginning of the line and the dollar sign ($) at the end of the line. Example:
^https://www.facebook.com/$
You can specify http AND https with the same rule by using regular expression quantifiers. Example:
^http[s]{0,1}://www.facebook.com/$
